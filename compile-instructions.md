# Compiling c / c++ Code using  gcc

## Compiling c

  ```
  gcc hello.c

  ```

  > Specify output file

  ```
  gcc -o hello hello.c

  ```

## Compiling C++

  ```
  g++ hwllo.cpp

  ```

  > Specify output file

  ```
  g++ -o hello hello.cpp

  ```
